JavaScript Array and string Methods

Array methods
1.The toString() method returns an array as a comma separated string:
2.The join() method also joins all array elements into a string
3.The pop() method removes the last element from an array.
4.The push() method returns the new array length.
5.The shift() method removes the first array element and "shifts" all other elements to a lower index.
6.The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements.
String methods.
1.The indexOf() method returns the index of the first occurrence of a specified text in a string
2.The lastIndexOf() method returns the index of the last occurrence of a specified text in a string
3.Both indexOf(), and lastIndexOf() return -1 if the text is not found.
4.The search() method searches a string for a specified value and returns the position of the match
